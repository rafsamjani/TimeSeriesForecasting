# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024

### Added
- Initial repository structure with organized directories
- Core forecasting methods:
  - ARIMA (AutoRegressive Integrated Moving Average)
  - LSTM (Long Short-Term Memory Neural Networks)
  - Prophet (Facebook's forecasting tool)
  - Traditional methods (Moving Average, Exponential Smoothing)
- Utility modules:
  - Data preprocessing functions
  - Visualization utilities
- Comprehensive documentation:
  - Main README with quick start guide
  - Getting Started tutorial
  - Methods comparison guide
  - Directory-specific README files
- Example scripts for all methods
- Jupyter notebook tutorial
- requirements.txt for dependency management
- .gitignore for Python projects

### Documentation
- Complete API documentation for utility functions
- Step-by-step examples for each forecasting method
- Academic usage guidelines for thesis/journal work
- Performance evaluation metrics explanation
- Method selection decision tree

### Examples
- `arima_example.py`: Complete ARIMA implementation
- `lstm_example.py`: Deep learning time series forecasting
- `prophet_example.py`: Business forecasting with Prophet
- `traditional_methods.py`: Baseline methods comparison

### Features
- Multiple evaluation metrics (RMSE, MAE, MAPE)
- Publication-ready visualizations
- Data preprocessing pipeline
- Model comparison framework
- Easy-to-extend architecture
