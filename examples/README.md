# Examples

This directory contains example scripts demonstrating how to use the Time Series Forecasting library.

## Available Examples

### example_usage.py

A comprehensive example that demonstrates:
- Generating synthetic time series data
- Testing for stationarity
- Training ARIMA and Prophet models
- Making predictions
- Evaluating model performance
- Comparing different models

## Running the Examples

To run an example:

```bash
cd examples
python example_usage.py
```

Make sure you have installed all dependencies first:

```bash
pip install -r ../requirements.txt
```
